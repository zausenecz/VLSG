Valorant Language Script Generator

One-click solution to change the voices in the VALORANT

Currently, only English is supported as the main language.
The voiceover language can be set to any of the available in the game.

Installation and setup instructions

Download files by clicking the download button

Unpack the archive and run the application with vlsg-update.exe

If additional packages need to be installed, accept and install using the links in the message

Follow the instructions in the app step by step

If an error occurs, check the correctness of the actions, if it does not help, please contact me in Telegram

After installation, use the VLSG shortcut to start the game

The shortcut should automatically appear in the start menu

If after updating the game, the game does not work properly or does not start at all, repeat the installation procedure to update the language packs
