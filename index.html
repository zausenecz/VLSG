Installation and setup instructions

Select the set of languages you want.

The options are currently supported:

  • English - Japanese
  • English - Russian

Configurations with a main language other than English do not work for some reason (the text is still in English).

Click the "Download" button and save the file anywhere.

Extract the downloaded archive.

Pay attention to where you have the game installed. If it is located on the default path on the C drive, you can skip this step. If the path is different, you need to specify it in the files.

To find out where the game is installed, open the Riot Client > SETTINGS > VALORANT. Note the line INSTALL PATH. It should match what is shown in the screenshot.

If not, open the downloaded folder, go to files > VLSG. Right click on the file Valorant.bat > edit. Change the paths to the following folders (check the screenshot).



Execute script.bat, press any key to continue the installation.

The VLSG folder will appear in the Documents folder and the shortcut of the same name will appear in the applications list.

Open the Riot Client settings again and select the desired voice language.

Wait until the download is complete.

After that, use the created shortcut to start the game. The first click will launch Riot Client, then click on the shortcut again and the game itself will start.

To undo the changes, change the language to your usual language and run the game through the standard shortcut.
